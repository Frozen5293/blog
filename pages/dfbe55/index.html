# 图形渲染管线

顶点处理：
    输入顶点
投影：
    变换顶点到屏幕上形成三角形，视图变换，
光栅化：
    分割成片元【像素】，深度测试，就是现实什么
着色：
    对像素颜色进行计算

现代的GPU是可以编程的，这意味着，***顶点***的着色与***像素***的着色由你说了算。

你的总结基本正确。如果你想实现一种非标准的透视效果，例如远大近小的效果，你可以自定义透视变换，而不是使用GPU自带的标准透视投影矩阵。这样，你可以手动计算变换前的结果并将其传递给片元着色器。在片元着色器中，你可以使用这些变换前的数据来计算光照和材质等效果。
